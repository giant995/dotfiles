# dotfiles

This is a repo of my most important dotfiles. It is a bad idea to fork that repo, but definetly dig in there to get inspired. Use at your own risks, nothing is guaranteed and I'm not responsible for any mess.

## Get started

This little install script is idempotent, meaning that it will not try to create symlinks and throw errors if one already exists.

1. Clone the repo into the home directory
2. cd into the folder
  ```bash
  cd ~/.dotfiles
  ```
3. Run the install script
  ```bash
  bash install.sh
  ```
